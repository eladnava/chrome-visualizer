# Installation

1. Copy the ChromeVisualizer directory to C:\Program Files\
2. Open the relevant Visual Studio sub-folder depending on your Visual Studio version. Follow the README.txt inside.
3. If you get a FileLoadException exception while attempting to open the Chrome Visualizer via the debugger context menu, you need to unblock the ChromeVisualizerIntegration.dll:
